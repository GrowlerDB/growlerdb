# Actors

The personas and principals that use GrowlerDB.

* [Platform admin / operator](/product/actors/platform-admin.md) - creates/operates indexes, configures access, runs the cluster
* [Search analyst / end user](/product/actors/search-analyst.md) - queries indexes and retrieves authoritative rows
* [Ingest operator](/product/actors/ingest-operator.md) - runs streaming ingestion, keeps indexes in sync
* [Application developer](/product/actors/app-developer.md) - embeds search into an app or retrieval pipeline
* [Service account](/product/actors/service-account.md) - non-human principal (API token) for programmatic access

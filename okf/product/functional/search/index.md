# Search

The read/search capabilities — everything a user does to find matching records.

* [Query](/product/functional/search/query.md) - full-text query via a canonical AST (Lucene/KQL forms)
* [Vector fields & embeddings](/product/functional/search/vector.md) - declare a VECTOR field; local-default embedding + per-doc vector storage for semantic / hybrid retrieval
* [Syntax (Lucene / KQL)](/product/functional/search/syntax.md) - choose the query grammar per request
* [Facets](/product/functional/search/facets.md) - aggregated value counts to filter and drill in
* [Suggest (autocomplete)](/product/functional/search/suggest.md) - type-ahead value suggestions
* [Sorting, paging & collapsing](/product/functional/search/sort-paging.md) - sort, offset/keyset paging, collapse, PIT
* [Highlighting](/product/functional/search/highlight.md) - query-term-highlighted snippets
* [Export](/product/functional/search/export.md) - export results to JSON or CSV

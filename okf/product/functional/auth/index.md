# Auth

Authentication and session capabilities. Authorization is [RBAC & tenancy](/product/functional/rbac-and-tenancy.md).

* [Login](/product/functional/auth/login.md) - authenticate via external IdP (OIDC/JWT) or built-in credentials
* [Logout & session revocation](/product/functional/auth/logout.md) - end a session; immediate revocation on role change / deprovision
* [API tokens](/product/functional/auth/tokens.md) - long-lived, revocable tokens for service accounts
* [mTLS](/product/functional/auth/mtls.md) - mutual-TLS transport authentication

# D36. License: AGPLv3

**Decision.** GrowlerDB's open-source distribution is licensed **AGPL-3.0-only**. AGPLv3 is OSI-approved
open source and keeps the engine and index store fully open; its network-copyleft ensures that
modifications offered to users over a network are shared back to the commons. A **commercial license**
is available for organizations that cannot use AGPL (for example, embedding GrowlerDB in a closed
product).

**Status.** Accepted. Supersedes the earlier Apache-2.0 intent (recorded in [D27](/system/decisions/d27-governance.md)).

# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

We pledge to make participation in the GrowlerDB community a harassment-free experience
for everyone. Be respectful, assume good faith, and help keep discussions
constructive.

Report unacceptable behavior to the maintainers via the contact in `SECURITY.md`
(or, until then, by opening a confidential issue). Maintainers will review and act
on reports in good faith.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
